条件语句

  • 如果条件语句中作为判断条件的是一个整形，则它的值为0时等同于布尔型的False，其余为True
  • 布尔型操作符xor
  • in操作符：检索某个对象容器中是否存在某一个对象（e.g.list）
  • is比较实例本身（cmp双等号比较内部变量的值）
    比如x=1y=1时 x is y是False但是 x==y是True
  • not对表达式进行取反
  • for除了直接标定次数以外还可以用range和xrange；前者返回一个新的数字list，包含指定范围内的数字；而后者返回一个迭代器，这种方式效率更高。
    - 二者下标均从0开始
    - 前闭后开（起点，终点，步长）
    - for i in range():
  • while 条件：
    - 和循环语句同级可以使用else语句 每次循环开始阶段的判断表达式都可以认为是一个if语句。
  • 函数def name(x,y....)
    - ()内可以为空 在代码中直接调用即可，有几个未知数，在代码中相应应当输入多少值。
    - 返回值return+返回命令

###类class和对象object

  • 对象是对变量和函数的封装，是一个单一实体，对象从类中获得他们的变量和函数，类本质上是一个创建对象的模板
  • 可以以一个类为模板创建多个对象，也可以更改某个对象的某个变量。
  • init()函数：是类的初始化函数，默认可以只有self函数，也可以传入别的参数进行初始化。
