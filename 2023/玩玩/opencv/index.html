opencv用多少学多少

一月 19, 2023
  • machine learning

opencv是一种计算机视觉算法 可以对图像视频进行特定识别处理

基本知识

HSV色彩空间

Hue色调 在HSV色彩空间中取值为[0,360]在opencv中直接将色调除以2，得到[0,180]之间的值（适应八位二进制256个灰度级的存储和表示范围）
Saturation饱和度 灰度颜色中RGB相等 饱和度为0 其范围应当为[0,1] 为了适应8位图的256个像素级 需要将其映射为[0,255]
Value亮度 范围为[0,1]映射同上 亮度值越大 图像越亮 亮度为0时是纯黑色

基础语句

图片导入img = cv2.imread('name.png/jpg...')
尺寸rows,cols = img.shape[:2]
.shape可以输出图像大小
cv2.imshow('name',img)可以打印图片 其中name为exe左上名称 img为图像参数
一般imshow后面会跟一个cv2.waitKey() 会在一定时间内等待接收键盘上的一个值 括号内可填写参数delay 如果<=0会一直等待按键 取正整数表示等待按键的时间（单位：帧） 期间按键返回按键阿斯特码值

图像识别

判断图像内像素点是否在范围内

dst = cv2.inRange(src,lowerb,upperb)

其中dst为输出结果（形似数组），src为检查对象（数组/图像），lowerb表示范围下界，upperb表示范围上界，在范围内时dst对应值为255，反之为0.

颜色提取/显示特定颜色的值

H通道表示颜色 S通道和V通道的取值范围一般为[100,255] 当饱和度或亮度太低时 计算的色调可能就不是很可靠

  • 以红色为例
    红色：[0,100,100] 到 [10,255,255]

图像的几何变换

几何变换指将一幅图像映射到另一幅图像内的操作

图像缩放

dst = cv2.resize(src,dize[,fx,[fx[,interpolation]]])

其中dst表示输出模型 大小为dsize 或者通过src.size()、fx、fy计算得到 fx是水平方向缩放比例 fy是垂直方向缩放比例
src表示缩放图像
interpolation表示差值方式
如果指定参数dsize的值 则无论是否指定了参数fx和fy 都由dsize决定目标图像大小
size函数定义示例size = (int(cols*9),int(rows*0.5))调用时直接在resize中调用即可rst=cv2.resize(img,size)
差值
*shape中前行后列dsize中相反

翻转

dst = cv2.flip(src,flipCode)

其中flipCode：0为竖直翻转 1为水平翻转 -1为水平竖直翻转
